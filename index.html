Commands

Configuration

Check Git version

git --version

Set your name

git config --global user.name "<Your Name>"

Set your email

git config --global user.email "<youremail@example.com>"

Set your GitHub account

git config --global user.username "USERNAME"

Create a Repository

Make a new folder (aka directory)

mkdir <FOLDERNAME>

List the items in a folder

ls

Navigate into an existing folder (aka change directory)

cd <FOLDERNAME>

Create a new repository

git init

Clone

Clone a remote repository to your computer

git clone <URL>

Clone a local repository

git clone /path/to/repository

Clone a repository on a remote server

git clone username@host:/path/to/repository

Status, Add and Commit Changes

Check status of changes to a repo (also shows current branch)

git status

Add a file's changes to be committed

git add <FILENAME>

Add all file's changes

git add .

Remove/delete file from INDEX (staging area)

git rm <FILENAME>

Commit changes to the repo's history with a short description of the changes

git commit -m "<your commit message>"

View changes to files (since last commit)

git diff

View changes between two branches

git diff <SOURCE_BRANCH> <TARGET_BRANCH>

Remote Connections

Add remote connections

git remote add <REMOTENAME> <URL>

Set a URL to a remote

git remote set-url <REMOTENAME> <URL>

Pull in changes

git pull <REMOTENAME> <BRANCHNAME>

View remote connections

git remote -v

Push changes

git push <REMOTENAME> <BRANCH>

or

git push <REMOTENAME>

Push 'master' branch to 'origin' on GitHub

git push origin master

See changes to the remote before you pull in

git fetch --dry-run

Branches

Create a new branch

git branch <BRANCHNAME>

Move onto a branch

git checkout <BRANCHNAME>

Rename a branch you're currently on

git branch -m <NEWBRANCHNAME>

Create and switch to a branch in one line

git checkout -b <BRANCHNAME>

List the (existing) branches

git branch

Merge a branch into current branch

git merge <BRANCHNAME>

Tagging

Create tag

git tag <tag> <COMMIT ID>

e.g.

git tag 1.0.0 1b2e1d63ff

Get commit IDs

git log

Deletion

Delete a local branch

git branch -d <BRANCHNAME>

Delete a remote branch

git push <REMOTENAME> --delete <BRANCHNAME>

Log

View repository history

git log

View only the commits of a certain author

git log --author=username

View compressed version of log where each commit is one line

git log --pretty=oneline

View log as an ASCII art tree of all the branches, decorated with the names of tags and branches

git log --graph --oneline --decorate --all

View long for only files which have changed

git log --name-status

View manual page for more info on git log command options and usage

git log --help

Restore

Replace working copy with latest from HEAD

git checkout -- <FILENAME>

Useful Hints

Built-in Git GUI

gitk

Use colorful Git output

git config color.ui true

Show log on just one line per commit

git config format.pretty oneline

Use interactive adding

git add -i
